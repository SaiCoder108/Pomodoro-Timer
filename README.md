# Pastel Pomodoro

A premium-feel focus timer built for deep work—no install, no build, just open and lock in.  

**Try it live:** [https://pastelfocus.vercel.app/](https://pastelfocus.vercel.app/)  

## Highlights

- **Focus Timer Engine:** Pomodoro presets (25/5, 50/10, 45/15), circular animated progress, work/break auto-switch, soft transition cues, timer runs across tabs.  
- **Task Workflow:** bulk paste tasks, automatic line-to-task parsing, quick mark done/undo, completed tasks section, local persistence.  
- **Multi-Mode Dashboard:** Pomodoro timer, live clock, next-day countdown, weekly session streak view.  
- **Theme System:** one-tap palette cycling, full theme swaps, dynamic favicon, 5+ creative palettes.  
- **Luxury UI Details:** glassmorphism cards, animated load screens, ambient blobs, smooth transitions, responsive layout.  
- **Persistence:** localStorage saves themes, timer, tasks, session history, loader state.  
- **Stack:** React 18 (CDN), TailwindCSS (CDN), Babel Standalone, single-file architecture.
